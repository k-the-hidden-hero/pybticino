BTicino WebRTC Audio Test

How this works: This page creates a WebRTC connection to a BTicino intercom using the Netatmo signaling WebSocket. It generates a silent audio track (0 Hz oscillator) to activate the device's microphone. Without real RTP audio packets, the device sends silence.

Note: This is a standalone test page. For production use, see the bticino_intercom Home Assistant integration.
